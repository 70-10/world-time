# world-time

## Install

```
$ npm install --global 70-10/world-time
```


## Usage

```
$ world-time
┌─────────────────────────┬─────────────────────────┐
│ Locale                  │ Time                    │
├─────────────────────────┼─────────────────────────┤
│ Asia/Tokyo              │ 2017-05-17 14:05:42     │
├─────────────────────────┼─────────────────────────┤
│ America/Los_Angeles     │ 2017-05-16 22:05:42     │
├─────────────────────────┼─────────────────────────┤
│ America/New_York        │ 2017-05-17 01:05:42     │
├─────────────────────────┼─────────────────────────┤
│ Europe/London           │ 2017-05-17 06:05:42     │
└─────────────────────────┴─────────────────────────┘
```
